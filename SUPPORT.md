# Support

## Where to Ask for Help

- Use GitHub Issues for reproducible bugs and feature requests.
- Include your operating system, app version, and clear reproduction steps.

## Security Issues

Do not report vulnerabilities in public issues.

Use the private reporting instructions in [SECURITY.md](SECURITY.md).

## Scope

This repository is maintained on a best-effort basis. Support is focused on:

- bugs in the application
- packaging or installation problems
- documentation issues

General operating system support or third-party tool support may be out of scope unless the problem can be reproduced inside Dev Janitor itself.
